# Dashboards Branch - Realtime BI Layer

Interactive executive dashboard built using Snowflake Native Streamlit connected directly to analytics models.

## Dashboard Modules
- Revenue KPI Cards
- First vs Last Attribution Comparison
- 5-Years Time Series Trends
- Channel Breakdown
- Live Streamed Events Feed

## Data Source
Snowflake ANALYTICS schema

## Deployment
GitHub dashboard branch ↔ Snowflake Git Integration ↔ Native Streamlit App

## Features
- Dark theme UI
- Real-time refresh
- Native Snowflake authentication
- Version controlled dashboards
